# Brawl Stats Tracker

Track Brawl Stars player statistics directly in Home Assistant.
Add multiple players per entry — each appears as its own device.
Trophies, victories, EXP, brawler count and more, live from the official Supercell API.

**Requires a free API key from [developer.brawlstars.com](https://developer.brawlstars.com/#/account).**
